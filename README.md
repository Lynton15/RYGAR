# RYGAR Technologies Website

This project is the official website for RYGAR Technologies. It showcases the company’s products, services, internships, courses, projects, and contact information.

## Features
- Modern landing page
- Product, service, and course sections
- Internship and project pages
- Responsive navigation and contact information

## Run locally
Since this is a static website, the easiest way to view it is to serve the folder with a simple local server.

### Using Python
From the project folder, run:

```powershell
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Project structure
- index.html — homepage
- about.html — about us page
- services.html — services page
- products.html — products page
- courses.html — courses page
- internships.html — internships page
- contact.html — contact page
- assets/ — CSS, JavaScript, images, and data files
- components/ — reusable navbar and footer

## Notes
- Open the site through a local server rather than directly opening HTML files to avoid component-loading issues.
- The content has been updated based on the provided PDF brief for RYGAR Technologies.
